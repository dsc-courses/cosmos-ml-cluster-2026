notes on setting up kaggle:

- go to kaggle, make a new restricted competition (this creates a URL that you can share)
- set subtitle to "See the starter notebook for instructions." since we don't want to repeat instructions twice and the starter notebook is supposed to have everything.
- now, open the competition settings. you have to complete all of the checklist items before you can release the competition.
- replace Overview with placeholder content, delete all other sections from the competition overview
- replace Rules with placeholder content.
- upload dataset, replace dataset description with placeholder content. set the license to anything.
- Access -> get invitiation link to share with students (i just put them in the notebook). set team size to 3.
- set key dates to friday at 4pm for both end date and private leaderboard
- scoring: set daily submission limit to 8 (roughly 1 per 30 mins). i left the rest alone.
- evaluation metric: i chose accuracy, although it would make sense to try other metrics depending on class imbalance
- upload solution file (i used a 50/50 public-private split)
- upload sandbox submissions (i used a few baseline models)
- finally, you can launch the competition!
